## 도메인별 건당 수작업 시간 기준값

| 도메인 | 건당 시간 |
|---|---|
| 이메일/발송 | 10분/건 |
| 문서/결재 | 15분/건 |
| 데이터/분석 | 30분/회 |
| 구매/조달 | 20분/건 |
| HR/인사 | 15분/건 |
| 고객/영업 | 10분/건 |
| (기타) | 15분/건 |

## 자동화 후 잔여 시간 (개선 후 예상 시간)

| effort | 잔여 시간 |
|---|---|
| 낮음 | 약 5~10분/주 |
| 중간 | 약 10~20분/주 |
| 높음 | 약 20~30분/주 |

## 구축 소요 시간

| effort | 구축 시간 |
|---|---|
| 낮음 | 4~8시간 |
| 중간 | 8~20시간 |
| 높음 | 20~40시간 |

## 입력 단위 변환 규칙

```
일(day)  기준 → 주간 환산 표시 (×5 근무일)
               연간 환산 추가  (×260 근무일)

월(month) 기준 → 월간 표시 유지
                연간 환산 추가 (×12)

년(year)+횟수 기준 → 연간 총합·회당 동시 표시
                     ROI 달성은 "약 N회차"로 표현

주(week) 기준 → 주간 표시 유지 / 연간 ×52
```

## 시나리오 기준값 (weekly_hours = null 시)

| 시나리오 | 주간 기준 | 설명 |
|---|---|---|
| 보수적 | 1시간 (60분) | 단순 반복 업무 |
| 중간 | 3시간 (180분) | 일반적인 수작업 |
| 적극적 | 5시간 (300분) | 복잡한 수작업 |

## ROI 달성 시점 표현 규칙

```
입력 단위가 주/일 → "약 N주" 또는 "약 N일"
입력 단위가 월    → "약 N개월"
입력 단위가 년/회 → "약 N회차"
달성 기준: 구축소요시간(분) ÷ 주간절감시간(분) = 주수
```
