# MS 솔루션 블랙리스트

Quick/Deep 모드 모두에서 AI 제안 직후 즉시 차단하는 사용 불가 목록.
AI가 이 목록에 포함된 제품을 제안한 경우 제안에서 즉시 제거한다.

---

## 차단 기준

| 코드 | 이유 |
|---|---|
| B001 | 서비스 deprecated — MS 공식 종료 |
| B002 | 회사 정책으로 사용 불가 |
| B003 | 운영 중 문제 확인 — 심각한 기능 결함 또는 보안 이슈 |

---

## 관리 원칙

- **추가 기준**: 실제 운영 중 문제가 확인된 경우에만 추가
- **추가 금지**: 단순히 solutions.md에 없다는 이유로 추가 금지
- **검토 주기**: 6개월마다 B002 항목 재검토 (정책 변경 여부 확인)
- **제거 조건**: deprecated 해제 또는 정책 변경 시 목록에서 제거

## 차단 적용 방법

AI 제안 후 blocklist.md 체크:

```
제안된 각 솔루션의 제품 목록을 순회
  → 이 목록에 포함된 제품이 있는가?
    YES → 해당 솔루션 즉시 제거 + 사유 표시
          예) "⛔ [제품명] — B001: deprecated (대체: [대체 제품])"
    NO  → 통과, 다음 단계 진행
```
